Face Recognition System

This project is a real-time Face Recognition System developed using Python and OpenCV. It utilizes deep learning techniques and Haar Cascade Classifiers to detect and recognize faces from a webcam feed.

Language Python
Libraries OpenCV, NumPy, face_recognition
Features
  • Live face detection through webcam
  • Face recognition using deep learning embeddings
  • Automatic marking of known and unknown users
  • Highly accurate and fast processing

This system can be integrated into security applications or attendance systems and works efficiently with multiple face profiles.
